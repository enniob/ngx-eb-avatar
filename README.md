# ngx-eb-avatar
ngx-eb-avatar is a lightweight avatar generator component.

## Install
```shell
npm install --save ngx-eb-avatar
```

## Usage
``` html
    <ngx-eb-avatar name="Your Name" 
                  [shadow]="true" 
                  width="50" 
                  height="50" 
                  bgColor="337ab7" 
                  fontColor="FFFFFF">
    </ngx-eb-avatar>
```
